### Setup

1. Click [SCAN](https://bit.ly/princerudh-md-qr) and scan QR through Whatsapp Linked Devices Option in Your whatsapp App.
2. You will get a session id in whatsapp, copy id only.
3. If You don't have a account in [Heroku](https://signup.heroku.com/), Create a account.
4. Click [FORK](https://github.com/SUHAID-BROO/PAATHU/fork)
5. Now [DEPLOY](https://heroku.com/deploy?template=https://github.com/SUHAID-BROO/PAATHU)
